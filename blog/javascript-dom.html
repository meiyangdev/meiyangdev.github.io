1. difference between HTML and CSS

HTML is Hypertext Markup Language and used to edit the content on the webpage. CSS stands for Cascading Style Sheet Language and used to add style to the content on the webpage.

HTML is like the skeleton of the human body, while css is like providing looks and appearence to the body.

control flow and loops is like building intelligence and logic into webpage. They enable computer to follow the scipts and make decisions and repeat tasks when specific criteria are met.

3.What is DOM? How do we interact with it?

DOM is document object model and it acts as a programming interface to access HTML and defines the logic structure of documents and the way a document is accessed and manipulated. DOM views an html document as a tree of nodes. Nodes is elements in html (eg head, body).

we can use developer tools like chrome browser and also code editor like VS to interact with it. We can create, change, move elements by using the javascipt language. We can also add events to these elements to make pege interactive.

4. Difference between accessing data from arrays and objects.

Array creates and store a list of data in a single varible while objects represents "things" with properties, in which is made of key and values. Arrays is zero-based indexing. We can use both dot and bracket notation to access, add and change and remove iterms in objects and arrays. We can also iterate over array iterms and objects properties by using loops.
